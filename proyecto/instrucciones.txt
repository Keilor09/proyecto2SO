1. Asegurarse de instalar gRPC Tools para python
pip install grpcio grpcio-tools

2. Asegurarse de tener instalado Python 3.10.12 en Linux

3. Para ejecutar el programa server.py se debe poner el siguiente comando:
python3 server.py

4. Una vez iniciado el server se puede utilizar el programa de client.py con el siguiente comando:
python3 client.py

5. El programa únicamente soporta texto, no se pueden imágenes o ningún otro tipo de archivo.
